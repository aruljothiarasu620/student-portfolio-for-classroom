# Student Portfolio for Classroom 🎓

A full-stack student portfolio application built with **React (Vite)**, **Node/Express**, and **SQLite/MySQL**.

## 🚀 Features
- **Student Dashboard**: Create and edit your personal profile.
- **Project Showcase**: Display your coding projects with links and images.
- **Skill Tracking**: Add and manage your technical skills.
- **Class Views**: Browse portfolios created by fellow students.
- **Secure Access**: JWT-based authentication for students and teachers.

## 🛠️ Setup & Running

### 1. Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## ☁️ Deployment (Vercel)
- **Frontend**: Connect the `frontend/` folder to Vercel (framework: Vite).
- **Backend**: Connect the `backend/` folder to Vercel (framework: Node.js).
- *Note: For persistent data on Vercel, switch from SQLite to a managed MySQL/PostgreSQL (Supabase, Neon, etc.) and update the `.env` variables.*
